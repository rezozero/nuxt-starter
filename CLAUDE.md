## Translations (i18n)

Translation keys are managed via **Xilofone**, an external translation management service.

The `i18n/locales/nuxt.*.json` files are automatically generated by the following command:

```bash
pnpm xilo
```

If a translation key needs to be added or updated, instruct the developer to make the change directly in the **Xilofone interface**, then run `pnpm xilo` to sync the local files.
